Class ScoreApi

ScoreApi provides an interface for configuring everything related to keeping track of the score of the game. This includes a "fact" interface for holding information about the current level, session, and game, setting the conditions for how a level is won or lost, tracking hero progress, managing timers, and interfacing with the counters for the number of goodies that have been collected.
