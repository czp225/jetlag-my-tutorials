Class ParallaxLayer

A ParallaxLayer is a layer that seems to scroll and repeat at a velocity that gives a sense of depth.

      The height of the image being used as a background layer

    • speed: number

      Speed at which it scrolls. Important values are 0, 1, and between. Differs for auto and non-auto.

    • isX: boolean

      True for X scrolling, false for Y scrolling

    • isAuto: boolean

      True if this should scroll regardless of camera

    • imgName: string

      The name of the image file to use as the background

    • config: JetLagConfig

      The game-wide configuration

    • device: JetLagDevice
