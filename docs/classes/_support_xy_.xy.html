Class XY

It's inconvenient to type PhysicsType2d.Vector2 every time we need it, and we aren't able to cleanly import the PhysicsType2d namespace. Since Vector2 is a pair of numbers, an easy workaround is to make this XY class, which is a name wrapper around PhysicsType2d.Vector2.
