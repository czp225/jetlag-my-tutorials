Class PointToActorCallback

PointToActorCallback is used by PhysicsType2d to determine if a coordinate in the world corresponds to a BaseActor.

ReportFixture

  • ReportFixture(fixture: Fixture): boolean
    • Defined in support/PointToActorCallback.ts:35

    ReportFixture is used by PhysicsType2d to see if the fixture that was found meets the requirements we establish. If so, it returns false, and the search ends, with user cod knowing it can find the result via getFoundActor(). If not, it returns true, and the search continues.

    Parameters

    • fixture: Fixture

      The fixture that was found, and which needs to be tested

    Returns boolean

    True if the fixture isn't satisfactory, and the search should continue
